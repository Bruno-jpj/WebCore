Dove si trova la root del progetto, dove si trova il file manage.py

> gunicorn nome_progetto.wsgi -> gunicorn project.wsgi

> gunicorn project.wsgi --workers 4 --bind 0.0.0.0:8000

I workers si calcolano così = 2 + CPU[core virtuali] + 1; non troppo alto questo numero;

Before deploying
> python manage.py check --deploy
